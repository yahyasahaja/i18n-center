# i18n-center Frontend

Next.js admin UI for centralized i18n management.

## Setup

1. Install dependencies:
```bash
yarn install
```

2. Set up environment variables:
```bash
cp .env.example .env.local
```

3. Run development server:
```bash
yarn dev
```

4. Build for production:
```bash
yarn build
yarn start
```

## Features

- Authentication with role-based access control
- Application management
- Component management
- Translation management with deployment stages
- Auto-translation using OpenAI
- Export/Import functionality
- User management (Super Admin, User Manager)

## Tech Stack

- Next.js 14
- React 18
- Redux Toolkit
- TailwindCSS
- TypeScript
